# 🚀 AI Web App – Text & Image Generation

This project is a prototype web application that demonstrates integration of multiple AI APIs to perform text enhancement and image generation workflows.

---

## 📌 Features

### 🔤 Text → Image Workflow
- User enters a text prompt
- System enhances the prompt using AI
- User approves the enhanced prompt
- AI generates an image based on the prompt

### 🖼️ Image → Analysis Workflow
- User uploads an image
- System analyzes the image using AI
- Returns a description/caption of the image

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Axios

### Backend
- Node.js
- Express.js

### APIs Used
- OpenAI API (Text enhancement, Image generation, Image analysis)
